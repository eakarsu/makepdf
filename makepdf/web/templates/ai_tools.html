AI-assisted PDF features (classification, summarization, PII redaction, form-fill mapping). All endpoints require the X-API-Key header and return 503 when OPENROUTER_API_KEY is not configured.

Stored in browser localStorage.

Classify Document

POST /api/classify — tags a PDF as contract / invoice / resume / medical / court_filing / report / manual / letter / other.

Summarize PDF

POST /api/summarize — extracts text + asks the LLM for a 2–4 sentence summary, key points, and named entities.

Smart Redact (PII)

POST /api/smart-redact — AI + regex PII detection. Use Preview for a dry run; otherwise the redacted PDF is downloaded.

Return proposed spans without writing redactions.

AI Form Fill

POST /api/ai-fill-form — map source JSON data to the AcroForm fields of a blank PDF form. Use Preview to see the mapping before filling.

Audit Log

Every AI call above is recorded. View paginated history at /ai-results.
